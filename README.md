# _JavaScript 30_ challenge by Wes Bos

## About the Challenge:

30 Day Vanilla JS Coding Challenge
Build 30 things in 30 days with 30 tutorials:

- No Frameworks
- No Compilers
- No Libraries
- No Boilerplate

Challenge link [HERE](https://javascript30.com/) if you want to take part.
